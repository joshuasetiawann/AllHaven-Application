# v3.3.0 — AllHaven 3.3 Routine Planner and Sidebar Flow

**Date:** 2026-06-13 · **Type:** Product UX release · **Product version:** `AllHaven 3.3`.

## Summary

AllHaven 3.3 replaces the old Calendar surface with a clearer **Routine** planner.
Tasks remain focused on work items and checklists, while Routine is now the place
for schedules, daily routines, time blocks, and plans with exact dates and hours.

## Added

- New `/dashboard/routines` page with a polished planner layout:
  - seven-day calendar strip at the top,
  - Today, Upcoming, Selected day, and All views,
  - routine stats for today, upcoming items, and all-day blocks,
  - next-routine summary,
  - jump-to-date control for planning today or far ahead,
  - complete create/edit modal with title, notes, place/context, start time, end time, and all-day toggle.
- New `/api/v1/routines/events` backend alias backed by the existing local event storage, so old data stays readable.
- `/dashboard/calendar` now redirects to `/dashboard/routines` so old bookmarks still work.

## Changed

- Sidebar primary order is now:
  `Dashboard`, `AI Chat`, `Routine`, `Task`, `Finance`, `Notes`, `Approval`.
- Dashboard quick actions now include Routine as a first-class workspace entry.
- AI section context treats Routine as the scheduling/time-blocking module.
- AI tool labels now describe schedule tools as routine items while preserving the existing internal tool names for compatibility.
- Approval copy now refers to Routine instead of Calendar for schedule write actions.

## Compatibility

- Existing `calendar_events` data and `/api/v1/calendar/events` endpoints remain available.
- The new Routine UI uses `/api/v1/routines/events`, which delegates to the same service and table.
