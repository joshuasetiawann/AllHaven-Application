# AllHaven 3.6.0

Release date: 2026-06-17

## Summary

AllHaven 3.6 is a small housekeeping release. It removes a personal local-path
identifier left in the in-repo development notes so the project ships clean as a
personal project. There are **no application code, API, schema, or behavior
changes** — every module works exactly as it did in 3.5.

## Changed

- **Privacy cleanup of development docs.** `docs/SESSION_MEMORY.md` contained a
  hard-coded personal home path (`/home/<user>/Desktop/.../CORE-OS-APPLICATION`)
  quoted from an old session brief. It is replaced with a generic placeholder
  (`~/CORE-OS-APPLICATION`) and the surrounding sentence reworded to "membuat
  path lokal tersebut". The note's meaning is unchanged.

## Not changed

- No backend, frontend, database, or configuration changes.
- No dependency updates.
- No new or removed features.

## Upgrade notes

Nothing to do. Pull the release; no migrations, rebuilds, or config edits are
required.
