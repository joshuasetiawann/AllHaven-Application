# v3.0.0 — AllHaven 3.0 Launch-Ready AI Workspace

**Date:** 2026-06-13 · **Type:** Major release · **Product version:** `AllHaven 3.0`.

This release ships **AllHaven 3.0**: a cleaner, more launch-ready app shell with
stronger AI Knowledge access, real app-level preferences, and no browser-native
decision popups.

## What changed

### App shell
- Refined the persistent sidebar width, rhythm, active states, icon wells, badges,
  and profile/footer area.
- Added clearer Workspace and Modules navigation groups.
- Improved the New Command button so the main action is easy to find without
  crowding the navigation.
- Adjusted AppShell spacing and content width for desktop, laptop, tablet, and
  phone.

### Top navigation
- Polished the sticky topbar with a cleaner command search trigger.
- Refined the Local AI status chip.
- Reworked notification/approval popover styling so pending actions are easier to
  scan.
- Framed settings and avatar actions so the right-side controls feel consistent.

### Shared UI system
- Updated cards, buttons, icon buttons, inputs, panel shadows, hover states, and
  focus treatment for a more consistent product feel.
- Improved page headers with a cleaner title/subtitle hierarchy and a subtle
  separator.
- Polished the command palette to match the redesigned topbar/sidebar language.

### Dashboard
- Added a stronger workspace header with version/status badges.
- Added quick entry points for Approvals and AI Chat.
- Refined stat tiles and card hierarchy for easier scanning.

### Settings and personalization
- Added device-local language controls for Bahasa Indonesia, English, and
  Traditional Mandarin.
- Added dark/light theme switching.
- Added accent color choices so the app can use Cyan, Emerald, Violet, or Amber
  as its primary UI nuance.
- Preferences apply globally through root theme tokens instead of living only in
  the Settings form.

### AI Knowledge
- AI Knowledge now contributes a lightweight document inventory to every chat
  context packet.
- Relevant indexed chunks are retrievable in every thinking mode, including Fast
  mode, so all selected models can use uploaded knowledge when it matches the
  prompt.
- Metadata-only files remain honest: the AI can say the file exists but must not
  claim to know unavailable binary contents.

### App popups
- Replaced browser `alert`, `confirm`, and `prompt` dialogs with AllHaven modal
  decisions and toast notifications.
- Covered AI conversation rename/delete, group create/rename/delete, AI Memory
  delete/clear, AI Knowledge delete, section memory clear-all, and Clock alarms.

## Database migration

No database migration is required for this patch.

## Verification

Recommended after pulling:

```bash
python3 -m compileall backend/app
```

If Node.js is available:

```bash
cd frontend
npm run build
```
